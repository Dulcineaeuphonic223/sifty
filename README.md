## What is sifty?

Sifty clears junk files from your Windows computer. It helps you regain space and improves system speed. You can look through your files before you delete them. The tool safely moves items to your Recycle Bin so you keep control of your data.

## 🛠 Features

Sifty provides a clean interface for managing your system storage.

- **Automated Scanning:** The tool finds temporary files, cached data, and old logs that take up space.
- **Visual Interface:** Use the simple text-based menu to navigate your file system safely.
- **Dry-Run Mode:** The software runs in a test mode first. It shows you what it wants to remove without actually deleting anything. You confirm the actions before any changes happen.
- **Recycle Bin Integration:** All deleted items go to the Recycle Bin. You can restore them if you change your mind.
- **Undo Capability:** If you decide a removal was a mistake, use the built-in undo command to bring back your files instantly.

## 🔍 How to use the clean tools

Once you start the program, a window appears with a menu. Use your keyboard to select options.

1. **Scan your drive:** Select the scan option to look for junk files. The tool displays a list of found items on your screen.
2. **Review your files:** Read through the list. Sifty marks files it considers safe to remove. You can choose to skip files you want to keep.
3. **Execute the clean:** Press the designated key to perform the cleanup. Because of the default safe settings, nothing enters the trash until you trigger this command.
4. **Check the results:** Open your Recycle Bin if you have doubts. If you need a file back, right-click the file and select Restore.

## ❓ Frequently Asked Questions

**Does this tool slow down my computer during a scan?** 
No. Sifty uses very few system resources. You can browse the web or work on documents while it checks for junk.

**Can I run this on my work computer?**
Most IT departments allow tools that do not require an installation. Since sifty runs as a single file, it usually complies with strict workplace rules. Check with your manager if you feel unsure.

**What happens if I turn off my computer during a clean?**
Nothing bad happens. The process just stops. Because the software uses the Windows Recycle Bin, your files stay safe even if the power cuts out mid-process.

**Will this tool remove my photos or documents?**
No. Sifty only looks for temporary files in standard system locations. It ignores your personal folders like Desktop, Documents, and Pictures.

**How do I delete the software when I am done?**
Simply delete the .exe file you downloaded. Because the program does not save settings to your system registry, it leaves no traces behind.

## ⚖ Understanding the terms

Sifty uses a command-line interface. This means it uses text commands to communicate. You do not need to type these commands manually. Use the arrow keys and the Enter key to move through the menus. This design keeps the program small and fast. It avoids the heavy graphics that make other programs slow. 

By default, the software acts as a "Dry-run". This means it shows you the work but keeps your files exactly where they are. You must opt into "Live-run" mode to make actual deletions. This two-step process ensures you never lose data by accident. 

When you perform the clean, the program moves files to the Recycle Bin. This is the safest way to clear space. If your computer feels slower after a clean, it is likely because you just removed redundant files that the system constantly manages. It will recover its speed once it indexes the new, cleaner state of your storage drive. 

If you find a file that sifty marks as junk, but you need it for a specific program, use the "Ignore" feature. This tells the tool to leave that specific item alone during future scans.
